How to setup Kafka?

Hi everyone, in this blog I'll be demonstrating how to setup kafka in your machine, create topics, send data into them using producers and read using consumers.
Steps to setting up Kafka differs depending on your operating system.
  • For Apache Kafka, java version must be Java 11 or above. Check installed version using command java --version in terminal.
    Download Java in your machine if java --version gives error message.
For MacOS:
  • Download of Kafka from Binary Downloads section.
  • Extract contents from downloaded binary into appropriate folder location
In our illustration, we are going to extract to ~ location.
To start Kafka, we need to first start Zookeeper. Zookeeper syncs Kafka services and use meta-data to keep track of distributed systems in Kafka
Steps to start Zookeeper:
  • Using terminal, go to same location where Kafka is extracted.
  • Using command sh kafka_2.13-3.0.0/bin/zookeeper-server-start.sh kafka_2.13-3.0.0/config/zookeeper.properties
Steps to start Kafka:
  • In a new terminal window, go to same location where kafka is extracted.
  • Use following command sh kafka_2.12-3.2.0/bin/kafka-server-start.sh kafka_2.12-3.2.0/config/server.properties
Kafka Service is up and running in your terminal.
